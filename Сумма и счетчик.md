## Пример 1: числа, кратные 3 (от 1 до 15) и их количество

**Что делает:**
- Проходит числа от 1 до 15
- Если число делится на 3 → выводит `"X кратно 3"`
- Иначе выводит просто число
- Подсчитывает количество кратных 3
- В конце выводит фразу `"Кратных 3: N"`

**Код:**
```java
public class Practice {
    public static void main(String[] args) {
        System.out.println(numbers());
    }

    public static String numbers() {
        String result = "";
        int count = 0;
        for (int i = 1; i <= 15; i++) {
            if (i % 3 == 0) {
                result += i + " кратно 3\n";
                count++;
            } else {
                result += i + "\n";
            }
        }
        return result + "Кратных 3: " + count;
    }
}
```

1  
2  
3 кратно 3  
4  
5

7  
8  
9 кратно 3  
10  
11

13  
14  
15 кратно 3  
Кратных 3: 5

## Пример 2: нечётные числа (от 1 до 10) и их сумма

**Что делает:**
· Проходит числа от 1 до 10  
· Если число нечётное → выводит `"X (нечет)"` и прибавляет его к сумме  
· Иначе выводит просто число  
· В конце выводит сумму всех нечётных чисел  

**Код:**
```java
public class Practice {
    public static void main(String[] args) {
        System.out.println(numbers());
    }

    public static String numbers() {
        String result = "";
        int sum = 0;
        for (int i = 1; i <= 10; i++) {
            if (i % 2 != 0) {
                result += i + " (нечет)\n";
                sum += i;
            } else {
                result += i + "\n";
            }
        }
        return result + "Сумма нечетных: " + sum;
    }
}
```
#### Пример вывода в консоли
1 (нечет)  
2  
3 (нечет)  
4  
5 (нечет)  
6  
7 (нечет)  
8  
9 (нечет)  
10  
Сумма нечетных: 25
