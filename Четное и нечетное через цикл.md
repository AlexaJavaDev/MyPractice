## Четное и нечетное  
Тут мы пропускаем цифры от 1 до 10 через цикл, чтобы он показал,  
какие числа четные, а какие нечетные четные и нечетные числа.

    public static void main(String[] args) {
        System.out.println(Numbers());
    }

    public static String Numbers() {
        String result = "";
        for (int i = 1; i <= 10; i++) {
            result = result + i + " ";
Проходимся по циклу и обнавляем **result**, где значение **i** - это число, т.е число(**i**) **+** пробел (**" "**).

            if (i % 2 == 0) {
                result = result + " Четное\n";
            } else {
                result = result + " Нечетное\n";
            }
        }
        return result;
    }
Дальше просто смотрим следующее:
- если (if) **i** кратно 2 (определяем четность), то выводим **result** и напротив него слово " Четное\n",
- иначе (else) выводим **result** и напротив него слово " Нечетное\n",
