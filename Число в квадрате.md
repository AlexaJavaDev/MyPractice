# Числа и их квадраты

Простая программа на Java, которая выводит числа от 1 до 10 и их квадраты.  
Это один из моих первых самостоятельных кодов.

---

## Что делает код?

- Запускается метод `main`, который вызывает метод `Numbers()`
- В цикле `for` числа от 1 до 10:
  - считаем квадрат числа (`i * i`)
  - добавляем в результат строку вида:  
    `Число: X Квадрат: Y`
- В конце все строки выводятся в консоль.

```java
public static void main(String[] args) {
        System.out.println(Numbers());
    }
    public static String Numbers() {
        String result = "";
        for (int i = 1; i <= 10; i++) {
            int sq = i * i;
            result += "Число: " + i + " Квадрат: " + sq + "\n";
        }
        return result;
    }
```
